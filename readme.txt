DFMconv. Пояснения
------------------

Назначение
----------
Если открыть исходники написанные в Delphi 6/7 с помощью Delphi 5, могут
появится ошибки связанные с присутствием русских символов в файлах DFM.
Delphi 6 хранит "нестандартные" (русские) символы в кодах Unicode.
Эта утилита создана для того, чтобы решить проблему. Она заменяет в 
DFM-файлах коды Unicode на символы WIN1251 понятные Delphi5.
Также есть функция перевода двоичных DFM-файлов в текстовые.

Пример
------
Кусок текста DFM ver 6:
    Caption = #1053#1072#1081#1090#1080
Заменяется на текст:
    Caption = 'Найти'

Особенности
-----------
- пакетное преобразование файлов;
- создание резервных копий файлов;
- удобное управление клавиатурой.

Управление клавиатурой
----------------------
[Ins]           - добавить;
[Del]           - исключить;
[Ctrl]+[Del]    - исключить все;
[S]             - старт;
[L]             - лог;
