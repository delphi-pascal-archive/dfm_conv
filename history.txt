DFMconv. История изменений
--------------------------

Обозначения
-----------
[+] Нововведение
[-] Исправлена ошибка
[*] Изменение

Версия 3.1 (23 ноя 2004)
------------------------
[+] Клавиша [Esc] закрывает Лог.
[-] Двоичный файл dfm версии 6 не перекодировался.

Версия 3.0 (9 мар 2004)
-----------------------
[+] Добавлена возможность преобразования двоичных файлов DFM в текстовые.
[*] Появилась закладка настройки.
[*] Исчезла птичка "Конечный файл". Если не указан конечный файл (пустая
    строка), преобразованный файл будет сохранен вместо исходного.

Версия 2.0 (2 фев 2004)
-----------------------
[-] Полностью переработан алгоритм разбора и преобразования строк.
    Неправильно обрабатывались строки типа
    1. '#' (возникала ошибка),
    2. #13#10 (вставлялся перевод строки).
    Теперь управляющие символы с кодом меньше 256 остаются как есть.
[*] Настройки сохраняются в ini-файл, который создается в папке с
    программой. Чтобы не осталось мусора в реестре, прочитайте readme.txt
    для данной версии.

Версия 1.2
----------
[+] Добавлена кнопка "Искл Все". Удаляет все файлы из списка на странице
    "Много файлов". [Ctrl]+[Del] - сочетание клавиш для нее.
[-] Чекбокс "Создавать Bak-файл" часто игнорировался и bak все равно
    создавался.

Версия 1.1
----------
[+] Сохраняются свойства некоторых элементов управления:
    1. активная страница (Один файл/Много файлов);
    2. чекбокс "Конечный файл";
    3. папки всех диалогов открытия и сохранения;
    Сохраняются в реестр:HKEY_CURRENT_USER\Software\DFMconv.
[+] Добавлены хот кеи:
    [Ins] - добавить;
    [Del] - исключить;
    [S]   - старт;
    [L]   - лог;
[+] Добавлена птичка "Конечный файл". Теперь на странице "Один файл" можно
    не указывать конечный файл, тогда изменения будут сохранены в исходный.
[-] В строке статуса отображался результат конвертирования последнего файла.
    Если при конвертировании нескольких файлов возникала ошибка не в
    последнем из них, в статусе было написано "ОК". Теперь пишется общее
    количество ошибок.
[*] Имена файлов не помещающиеся по ширине в список, преобразуются к виду
    "C:\...ir\filename.ext".
[*] Поменял местами кнопки "Старт" и "Лог". Теперь левее находится более
    важная кнопка :)
